# App Installer Script for Windows

## HOW TO USE
**just run this on your powershell:**
```powershell
Set-ExecutionPolicy Bypass -Scope Process -Force;iwr -useb https://raw.githubusercontent.com/RizalAchp/rz-winscripts/master/install.ps1 | iex
```
